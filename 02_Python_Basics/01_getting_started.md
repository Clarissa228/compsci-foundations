# Getting Started with Python

## What is Python?

Python is a **high-level programming language** — meaning it reads almost like English, and you don't have to worry about telling the computer how to manage memory or hardware. It's one of the most popular languages in the world and a perfect first language.

Python is used in:
- Web development (Instagram, Pinterest use Python)
- Data science and AI (most ML frameworks support Python)
- Automation (scripting repetitive tasks)
- Scientific research

---

## Installing Python

1. Go to [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. Download the latest version (3.x — **not** 2.x)
3. Run the installer. **Important**: Check the box that says "Add Python to PATH"
4. Open a terminal (Command Prompt on Windows, Terminal on Mac/Linux)
5. Type `python --version` and press Enter — you should see something like `Python 3.12.0`

---

## Installing VS Code (Code Editor)

A code editor is where you'll write your programs. VS Code is free and excellent.

1. Download from [https://code.visualstudio.com/](https://code.visualstudio.com/)
2. Install it and open it
3. Install the **Python extension** (search "Python" in the Extensions panel on the left)

---

## Your First Program

Create a new file called `hello.py` and write this:

```python
print("Hello, world!")
```

Run it by typing `python hello.py` in the terminal (or clicking the Run button in VS Code).

You should see:
```
Hello, world!
```

That's a real program. You just wrote code. 🎉

---

## The Python Interactive Shell (REPL)

Python has an interactive mode where you can type code and see results instantly. Just type `python` in your terminal:

```
>>> 2 + 2
4
>>> print("test")
test
>>> 10 * 5
50
```

This is great for trying out small ideas quickly. Press `Ctrl+Z` (Windows) or `Ctrl+D` (Mac/Linux) to exit.

---

## Comments

A **comment** is a line of code that Python ignores — it's just a note for you (or future readers):

```python
# This is a comment - Python will not run this line
print("This line runs")  # You can also comment at the end of a line
```

Comments are incredibly important. Write them whenever you do something that might be confusing later.
