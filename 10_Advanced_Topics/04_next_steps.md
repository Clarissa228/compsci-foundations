## What to Build Next

The best way to grow as a programmer is to **build things**. Here are some project ideas at different levels:

### Beginner Projects
- A command-line calculator with memory
- A text-based quiz game
- A simple to-do list app (save to a file)
- A word count tool (reads a .txt file, shows top words)

### Intermediate Projects
- A password manager (encrypted file storage)
- A simple web scraper (download and analyze data from the web)
- A contact book with SQLite
- A text-based RPG game using OOP

### Advanced Projects
- Build your own HTTP server from scratch
- Build a mini-interpreter for a simple language
- Implement a compression algorithm
- Build a recommendation system

## Paths to Specialize In

CS is a vast field. Here are the main directions you can take:

### 🌐 Web Development
**What**: Build websites and web applications
**Learn next**: HTML/CSS, JavaScript, React (frontend); Flask/Django (backend)
**Resources**: The Odin Project (free, excellent), freeCodeCamp

### 📊 Data Science / ML
**What**: Analyze data, build predictive models
**Learn next**: numpy, pandas, matplotlib, scikit-learn, statistics
**Resources**: fast.ai, Kaggle, 3Blue1Brown's Linear Algebra series

### 🔧 Systems Programming
**What**: OS, compilers, embedded systems, performance-critical code
**Learn next**: C or C++, computer architecture, operating systems
**Resources**: CS:APP (Computer Systems: A Programmer's Perspective), MIT 6.004

### 🔒 Cybersecurity
**What**: Protect systems, find vulnerabilities
**Learn next**: Networking, Linux, cryptography basics
**Resources**: picoCTF, HackTheBox, OverTheWire wargames

### 📱 Mobile Development
**What**: Build iOS/Android apps
**Learn next**: Swift (iOS) or Kotlin (Android), or React Native for both
**Resources**: Apple Developer docs, Google developer docs

### 🤖 AI / Machine Learning
**What**: Build intelligent systems
**Learn next**: Linear algebra, statistics, PyTorch, deep learning
**Resources**: fast.ai, Andrew Ng's ML course, Andrej Karpathy's Zero to Hero

## Preparing for CS in College / Career

If you're thinking about studying CS or working in tech:

1. **Practice coding problems** — LeetCode, HackerRank, Exercism
2. **Build a portfolio** — 3–5 projects on GitHub that you're proud of
3. **Learn git** — version control is essential for any developer
4. **Take CS50** — it's free, rigorous, and deeply respected
5. **Study for interviews** — algorithms + data structures are the main focus

### Learn Git (Version Control)
```bash
git init                      # Start a repository
git add file.py               # Stage changes
git commit -m "My message"    # Save a snapshot
git push                      # Upload to GitHub
```

👉 [Git tutorial — The Odin Project](https://www.theodinproject.com/lessons/foundations-git-basics)
