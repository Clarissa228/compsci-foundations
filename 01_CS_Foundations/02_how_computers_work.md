# How Computers Work

## The Big Idea

A computer is a machine that processes **information**. Everything it does — playing music, browsing the web, running AI — comes down to manipulating tiny electrical signals at incredible speed.

---

## The Four Main Components

### 1. CPU — Central Processing Unit
The "brain" of the computer. It executes instructions — like doing math, comparing values, moving data. Modern CPUs run **billions of operations per second**.

### 2. RAM — Random Access Memory
Temporary memory. When you open a program, it gets loaded into RAM so the CPU can access it quickly. RAM is *fast* but *temporary* — it clears when you turn off your computer.

### 3. Storage (Hard Drive / SSD)
Long-term memory. Your files, photos, and installed programs live here. Much slower than RAM but it persists even when the computer is off.

### 4. I/O — Input / Output
How the computer communicates with the world:
- **Input**: keyboard, mouse, microphone, camera
- **Output**: monitor, speakers, printer

---

## How a Program Actually Runs

When you run a Python script, here's what happens step by step:

1. You write code in a text file (e.g. `hello.py`)
2. Python translates (interprets) your code into instructions the CPU understands
3. The CPU executes those instructions one by one
4. Results appear on your screen (output)

```
You write code → Python interprets it → CPU executes → Screen shows result
```

---

## What is an Operating System?

Your **OS** (Windows, macOS, Linux) is the software that manages everything. It:
- Decides which programs get CPU time
- Manages memory
- Handles your keyboard and mouse
- Keeps files organized on your disk

Without an OS, your computer would be a very expensive paperweight.

---

## Layers of a Computer System

Think of it like a building:

```
Layer 5: You (the user)
Layer 4: Applications (browser, Python, games)
Layer 3: Operating System (Windows, macOS, Linux)
Layer 2: Hardware Drivers (software that talks to hardware)
Layer 1: Physical Hardware (CPU, RAM, storage, screen)
```

As a programmer, you mostly work at Layer 4 — but understanding the layers below makes you a much better engineer.

## Key Takeaways

- Computers have **CPU** (processor), **RAM** (temporary memory), **storage** (permanent), and **I/O**
- Programs are translated into CPU instructions at runtime
- The **OS** manages all hardware and software on your behalf
- Understanding the layers helps you debug and write better code
