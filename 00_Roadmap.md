# 🗺️ Computer Science Foundations — Full Roadmap

Welcome! This course will take you from **zero coding experience** all the way to understanding the core ideas that power modern software. You'll use **Python** as your language throughout.

Take your time with each module. There's no rush — understanding beats speed every time.

---

## 📚 How This Course Works

Each module has its own folder. Inside each folder you'll find:
- **Topic files** (`.md`) — short, focused explanations with examples
- **`questions.md`** — practice problems to test your understanding
- **YouTube links** — video explanations built into each topic file

Work through the files in order. Try the questions *before* looking things up — struggling a little is how you actually learn.

---

## 💡 Tips for Learning CS

1. **Type out every code example** — don't copy-paste. Your fingers need to learn too.
2. **Break things on purpose** — change values, delete lines, see what happens.
3. **The questions are the most important part** — they show you where your understanding has gaps.
4. **Google is a tool, not a cheat** — every real programmer uses it constantly.
5. **Take breaks** — your brain consolidates learning while you rest.
